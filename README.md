# 📅 Calendar Event App

A clean and interactive calendar application built using React (Next.js) and Tailwind CSS.

## 🚀 Features
- Monthly calendar view
- Navigate between months
- Select a date
- Add, edit, and delete events
- Events stored using localStorage (persistent)
- Highlights today's date
- Clean and responsive UI

## 🛠 Tech Stack
- React / Next.js
- Tailwind CSS
- JavaScript

## ▶️ Run Locally
npm install
npm run dev
